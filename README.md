# Social Media & Mental Health Analysis  
This project explores the relationship between social media usage and mental health across all age groups using data visualization and analysis techniques. The goal is to understand how prolonged use of social media may impact sleep quality, stress levels, and overall happiness.  

## Project Structure  
Visualization/ – Contains visual outputs created using Tableau, illustrating key trends and correlations in the data.  
Report/ – A detailed project report including data cleaning, methodology, visual analysis, and references.  

## Project Overview  
Using a dataset from Kaggle, this project investigates the impact of daily social media use on three key mental health indicators:Sleep quality, Stress level, Happiness index.  
The analysis focuses on individuals of all ages and genders, with data cleaned to ensure authenticity (e.g., removal of incomplete entries and outliers).  

## Key Findings  
Sleep quality declines sharply with increased social media usage, likely due to screen exposure and cognitive arousal before sleep.

## Conclusion  
The findings suggest a consistent negative correlation between prolonged social media use and mental well-being across all age groups. These insights can inform mental health strategies, such as encouraging reduced screen time and promoting real-world engagement.

# 社交媒体与心理健康分析  
本项目通过数据可视化和分析技术，探索社交媒体使用与各年龄段人群心理健康之间的关系，旨在了解长期使用社交媒体对睡眠质量、压力水平和整体幸福感的影响。  

## 项目结构  
Visualization/ – 包含使用Tableau创建的视觉输出，展示数据中的关键趋势和相关性。  
Report/ – 详细的项目报告，包括数据清洗、研究方法、可视化分析和参考文献。  

## 项目概述  
本项目使用来自Kaggle的数据集，研究每日社交媒体使用对三个关键心理健康指标的影响：睡眠质量、压力水平、幸福指数。  
分析对象涵盖所有年龄段和性别的个体，数据经过清洗以确保真实性（比如删除不完整条目和异常值）。  

## 主要发现  
随着社交媒体使用时间的增加，睡眠质量显著下降，这可能和睡前屏幕暴露和认知唤醒有关。

幸福指数也有所下降，但幅度较小，这表明社交媒体仍可能提供一些积极的社交联系。

## 使用工具  
Tableau – 用于数据可视化和仪表板创建  
Python / Pandas – 用于数据清洗和预处理  
Kaggle – 数据集来源  

## 结论  
研究结果表明，在所有年龄段中，长时间使用社交媒体与心理健康状况之间存在持续的负相关关系。这些发现可为心理健康策略提供参考，例如鼓励减少屏幕时间和增加现实世界互动。
